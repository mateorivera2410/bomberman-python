# Bomberman Python

Bomberman-style game developed in Python as a programming project.

This project is a console-based version of Bomberman where the player can start a new game, move through a generated world, interact with objects, and advance through different levels.

## Features

- Main menu system
- New game option
- Random world generation
- Multiple levels
- Player movement
- Bomb mechanics
- Modular code using different Python files

## Technologies Used

- Python
- Functions
- Loops
- Conditionals
- Modular programming
- Basic game logic

## Project Structure


bomberman-python/
├── interface.py
├── logic.py
├── objects.py
├── README.md
├── requirements.txt
└── .gitignore


How to Run
Download or clone this repository.
Open the project folder.
Run the main file:
```txt
python interface.py
```

Files Description
main.py: Main file where the game starts.
logic.py: Contains the main logic of the game.
objects.py: Contains the objects used in the game.
README.md: Project explanation.
requirements.txt: Project dependencies.
.gitignore: Files and folders that should not be uploaded to GitHub.
